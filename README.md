#### \_post 目录下文章格式

开头
layout: 固定是 post
title: 列表页面标题位置，详情页面标题
categories: [cate1, cate2]，分类列表，可以定义多个
description: some word here，描述
excerpt：列表页面显示的副标题
keywords: keyword1, keyword2，搜索的关键词
